# Sistema de Gestão da Clínica Veterinária Vida Animal – Scripts SQL

Este repositório contém os scripts SQL referentes ao mini-mundo da clínica veterinária “Vida Animal”, desenvolvidos na disciplina de Modelagem de Banco de Dados.

O objetivo é criar, popular e manipular um banco de dados relacional utilizando comandos DDL e DML (CREATE, INSERT, SELECT, UPDATE e DELETE), aplicando normalização e integridade referencial.

## Tecnologias e Ferramentas

- Banco de dados: MySQL
- Ferramenta de desenvolvimento: MySQL Workbench
- Linguagem: SQL padrão

## Estrutura do Repositório

```text
sql/
